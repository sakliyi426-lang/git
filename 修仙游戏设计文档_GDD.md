## 1. 游戏概述

### 1.1 一句话定位

> **一款以"寿命"为核心压力、以"悟道"为核心驱动的开放世界修仙RPG。大世界自由探索（鬼谷式），战斗切入回合策略（觅长生式）。**

### 1.2 核心卖点（融合两者精华）

| 卖点 | 来源灵感 | 融合创新 |
|------|---------|---------|
| 🌍 **开放世界探索** | 鬼谷八荒 | 手工 + 程序生成混合地图 |
| ⚔️ **回合制策略战斗** | 觅长生 | 实时探索 → 回合战斗无缝切换 |
| 🔥 **功法自由搭配** | 觅长生 | 功法组合产生"道韵"协同效果 |
| 🧠 **悟道系统** | 觅长生 | 通过战斗、探索、冥想多种途径悟道 |
| 💊 **深度副职业** | 觅长生 | 炼丹/炼器/阵法/灵植/符箓五大体系 |
| 👥 **NPC社交网络** | 鬼谷八荒 | 师徒/道侣/仇敌/盟友关系链 |
| ⏳ **寿命与渡劫压力** | 鬼谷八荒 | 寿命是核心资源，倒计时驱动 |
| 🏛️ **宗门经营** | 鬼谷八荒 | 从弟子到掌门的完整成长线 |

### 1.3 核心设计哲学

```
┌─────────────────────────────────────────────────┐
│  "时间是最稀缺的资源，每一次选择都有代价"        │
│                                                   │
│  · 闭关修炼 → 实力提升，但寿命流逝               │
│  · 外出探索 → 获得机缘，但有死亡风险             │
│  · 交际应酬 → 积累人脉，但耽误修炼               │
│  · 炼丹炼器 → 赚取资源，但消耗材料和时间         │
│                                                   │
│  → 玩家必须不断做取舍，没有"完美路线"            │
└─────────────────────────────────────────────────┘
```

## 2. 世界观设定

### 2.1 世界背景

**苍穹界** —— 一个灵气复苏后又逐渐衰退的修仙世界。

- **上古时代**：大能辈出，仙人飞升 commonplace
- **当前时代**：灵气稀薄，天道法则混乱，飞升之路断绝
- **核心矛盾**：天地灵气枯竭 → 各势力争夺资源 → 修士之间尔虞我诈

### 2.2 地理结构

```
                    ┌─────────────┐
                    │   仙界(封印) │  ← 终极目标：重开飞升之路
                    └──────┬──────┘
                           │
            ┌──────────────┼──────────────┐
            │              │              │
     ┌──────┴──────┐ ┌────┴─────┐ ┌──────┴──────┐
     │  北方冰原   │ │ 中央九州 │ │  南方密林   │
     │ (魔修聚集)  │ │(凡人国度)│ │ (妖兽横行)  │
     └─────────────┘ └──────────┘ └─────────────┘
            │              │              │
     ┌──────┴──────┐                ┌─────┴──────┐
     │  西方大漠   │                │ 东方海域   │
     │ (散修游荡)  │                │ (龙宫秘境) │
     └─────────────┘                └────────────┘
```

### 2.3 修仙体系总览

```
凡人 → 练气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 仙人
 ↑                                                                  ↑
起点                                                               终点
(寿命80)                                                         (寿命无限)
```

### 2.4 势力阵营

| 阵营 | 特点 | 关系 |
|------|------|------|
| **正道宗门**（剑宗、丹宗、符宗等） | 讲规矩、重传承 | 与魔道对立 |
| **魔道势力**（血煞门、幽冥教等） | 不择手段、实力强 | 与正道对立 |
| **散修联盟** | 松散、自由、无拘束 | 中立摇摆 |
| **妖族**（妖兽、妖修） | 肉身强悍、寿命长 | 看部落而定 |
| **凡人国度** | 无修为但人多 | 被修士影响 |

## 3. 核心玩法循环

### 3.1 主循环（Core Loop）

```
    ┌───────────────────────────────────────────────────────┐
    │                                                       │
    ▼                                                       │
  ┌─────────┐     ┌──────────┐     ┌──────────┐          │
  │ 修炼闭关 │────▶│ 实力提升  │────▶│ 探索秘境  │          │
  │ (消耗时间)│     │ (境界↑)   │     │(获取资源) │          │
  └────┬─────┘     └──────────┘     └─────┬────┘          │
       │                                   │               │
       │           ┌──────────┐            │               │
       │           │ 战斗(回合)│◀───────────┘               │
       │           └────┬─────┘                            │
       │                │                                   │
       │           ┌────▼─────┐                            │
       │           │ 获取战利品│                            │
       │           │/受伤/结仇 │                            │
       │           └────┬─────┘                            │
       │                │                                   │
       │     ┌──────────┴──────────┐                       │
       │     ▼                     ▼                       │
       │  ┌────────┐          ┌─────────┐                  │
       └─▶│炼丹炼器│          │ 社交互动 │                  │
          │(强化自身)│          │(人脉/情报)│                 │
          └────────┘          └────┬────┘                               │
                                   │                     │
                              ┌────▼────┐                │
                              │悟道突破  │─────────────────┘
                              │(境界提升)│
                              └─────────┘
```

### 3.2 寿命驱动的紧迫感

```
时间流逝机制：
┌─────────────────────────────────────────┐
│  · 闭关1次     = 消耗 1~10年            │
│  · 探索1个区域 = 消耗 数月~1年          │
│  · 炼制1炉丹药 = 消耗 数天~数月         │
│  · 社交1次事件 = 消耗 数天~数周         │
│                                          │
│  每个境界有寿限：                        │
│  练气: 120年  筑基: 200年  金丹: 500年 │
│  元婴: 1000年 化神: 2000年 ...          │
│                                          │
│  → 寿命归零 = 死亡(可转世重修)          │
│  → 渡劫失败 = 身死道消(可选重修)        │
└─────────────────────────────────────────┘
```

### 3.3 一局游戏的时间跨度

| 阶段 | 境界 | 游戏内时间 | 玩法重心 |
|------|------|-----------|---------|
| 新手期 | 练气 | 1~50年 | 学习基础、加入宗门 |
| 发展期 | 筑基 | 50~200年 | 探索地图、学副职 |
| 中期 | 金丹 | 200~500年 | 秘境探索、结交道侣 |
| 深度期 | 元婴 | 500~1000年 | 宗门经营、渡劫准备 |
| 终局 | 化神+ | 1000年+ | 挑战天道、寻求飞升 |

## 4. 角色系统

### 4.1 灵根系统（先天资质）

```
灵根类型（决定修炼速度和功法适配）：
┌──────────────────────────────────────────────┐
│  五行灵根：金·木·水·火·土                     │
│  异变灵根：雷·风·冰·毒                        │
│  特殊灵根：剑灵根·混沌灵根·天道灵根(极稀有)  │
├──────────────────────────────────────────────┤
│  天灵根(单属性) ─ 修炼最快，悟性加成          │
│  真灵根(2~3属性) ─ 较快，主流选择             │
│  伪灵根(4~5属性) ─ 慢，但有独特潜力           │
│  废灵根(无) ─ 困难模式，机缘可逆转            │
└──────────────────────────────────────────────┘

灵根品质：下品 < 中品 < 上品 < 极品 < 仙品
```

### 4.2 核心属性

| 属性 | 说明 | 影响 |
|------|------|------|
| **气血(HP)** | 生命值 | 归零则死亡 |
| **灵力(MP)** | 施法资源 | 释放功法消耗 |
| **神识** | 精神力 | 影响悟道、抗幻术 |
| **攻击** | 伤害基础 | 物理功法伤害 |
| **法术** | 法术伤害 | 五行功法伤害 |
| **防御** | 减伤 | 降低受到伤害 |
| **速度** | 行动顺序 | 回合制中的先手 |
| **闪避** | 躲避概率 | 实时模式的操作容错 |
| **悟性** | 学习效率 | 功法领悟速度、悟道概率 |
| **气运** | 运气 | 掉落率、随机事件倾向 |
| **寿命** | 剩余生命年限 | 归零则寿终 |

### 4.3 属性成长机制

```
基础属性 = 先天灵根基础值
         + 境界突破加成
         + 功法加成（装备的修炼功法）
         + 装备加成（法器/法宝）
         + 丹药加成（临时/永久）
         + 悟道加成（道韵效果）

→ 角色build多样性来源：功法 + 悟道 + 装备的搭配
```

### 4.4 心境系统（鬼谷特色强化）

```
心境值 (0~100)：
  · 高心境 → 悟道概率↑，走火入魔概率↓
  · 低心境 → 走火入魔风险↑，修炼效率↓

心境影响因素：
  + 战斗胜利、突破成功、悟道、帮助他人
  - 战斗惨败、亲人死亡、杀戮过多、被算计

特殊状态：
  · 走火入魔 → 属性暴增但失控，可能永久受伤
  · 心魔劫   → 元婴境以上突破时的心境考验
```

## 5. 战斗系统（混合制）

### 5.1 设计理念

```
┌─────────────────────────────────────────────────────────┐
│                                                           │
│   大世界探索（实时）           战斗触发（无缝过渡）       │
│   ┌─────────────────┐          ╲                         │
│   │  · 自由移动      │           ╲                       │
│   │  · 遇敌可视      │────▶  ┌─────────┐                 │
│   │  · 走位躲避      │       │ 回合制   │                 │
│   │  · 环境互动      │◀────  │ 战斗场景 │                 │
│   │  · 偷袭/逃跑     │       └─────────┘                 │
│   └─────────────────┘          ╱                         │
│                              ╱                           │
│                          战斗结束                        │
│                                                           │
│  → 实时模式 = 探索、选择、定位（鬼谷体验）              │
│  → 回合模式 = 策略、搭配、博弈（觅长生体验）            │
└─────────────────────────────────────────────────────────┘
```

### 5.2 实时探索模式

```
操作：
  WASD / 方向键 ── 移动
  空格 ────────── 互动/拾取
  Tab ─────────── 切换目标
  E ───────────── 使用快捷道具

遇敌机制：
  · 怪物在大地图上可见（明雷遇敌）
  · 碰触怪物 → 触发战斗
  · 从背后接触 → 获得先手优势（偷袭）
  · 从正面接触 → 正常战斗
  · 等级差距大 → 可选择"威压"直接吓退（消耗神识）

逃跑机制：
  · 实时模式中远离怪物一定距离 = 脱战
  · 回合制中可选择"逃跑"指令（成功率看速度差）
```

### 5.3 回合制战斗详解

#### 5.3.1 行动顺序（ATB 变体）

```
速度条机制（时间轴）：
  
  每个角色有独立"行动槽"，按速度值填充：
  
  角色A(速100): ████████████████▓▓ → 即将行动
  角色B(速80):  ████████████░░░░░░ 
  敌人C(速60):  ██████████░░░░░░░░ 
  
  → 速度越快，行动频率越高
  → 某些功法可以加速/减速行动槽
```

#### 5.3.2 灵力管理系统（核心策略）

```
灵力(MP)是战斗核心资源：

  · 每回合自然恢复 X 点灵力（看功法）
  · 释放功法消耗灵力
  · 灵力不足 → 只能用普通攻击
  · 灵力过载 → 可以强行爆发但有反噬风险

五行相生相克：
  金生水·水生木·木生火·火生土·土生金
  金克木·木克土·土克水·水克火·火克金
  
  · 用克制属性的功法 → 伤害+50%
  · 用相生属性连招   → 触发"道韵共鸣"额外效果
```

#### 5.3.3 功法槽与技能组合

```
战斗中可装备的功法槽位：

  ┌─────────────────────────────────┐
  │ 主动功法槽 ×4  （战斗中主动释放）│
  │  [烈火掌] [冰封万里] [金锋剑] [土遁]│
  ├─────────────────────────────────┤
  │ 被动功法槽 ×2  （自动生效）      │
  │  [火灵体] [剑心通明]            │
  ├─────────────────────────────────┤
  │ 遁术槽 ×1     （逃跑/位移用）    │
  │  [土遁术]                      │
  ├─────────────────────────────────┤
  │ 法宝槽 ×3     （主动使用，充能制）│
  │  [飞剑] [护身符] [混元珠]       │
  └─────────────────────────────────┘
```

#### 5.3.4 战斗指令

```
每回合可选行动：
  ┌──────────┬──────────────────────────────┐
  │ 攻击     │ 普通攻击（不耗灵力，伤害低） │
  │ 功法     │ 选择主动功法释放             │
  │ 法宝     │ 使用法宝（消耗充能）         │
  │ 道具     │ 使用丹药/符箓                │
  │ 蓄力     │ 跳过回合，灵力恢复加倍       │
  │ 防御     │ 减伤50%，灵力少量恢复        │
  │ 遁走     │ 尝试逃跑                     │
  └──────────┴──────────────────────────────┘
```

### 5.4 "道韵共鸣"系统（核心创新）

> **融合觅长生的功法连招 + 自创的元素反应机制**

```
连续使用特定组合的功法，触发额外效果：

火 + 风 = "烈焰风暴" → 范围灼烧 + 持续伤害
水 + 冰 = "冰封领域" → 冻结概率大幅提升  
金 + 土 = "金戈铁马" → 防御穿透 + 破甲
木 + 水 = "枯木逢春" → 回复气血 + 解除负面

连招追踪：
  · 系统记录最近3回合使用的功法属性
  · 匹配到组合 → 自动触发共鸣
  · 悟道等级越高 → 可解锁更多共鸣组合
```

### 5.5 战斗结果与风险

```
胜利 → 经验值、灵石、掉落物、声望变化
惨胜 → 重伤(临时属性下降，需疗伤)
失败 → 
  · 普通战败 → 损失物品，被丢回城镇
  · 深度战败 → 重伤需长期静养(消耗寿命)
  · 致命战败 → 身死(可读档/转世重修)
```

## 6. 修炼与功法系统

### 6.1 境界体系

```
┌────────┬──────┬────────┬──────────────────────────────┐
│ 境界   │ 寿命 │ 层级   │ 突破条件                     │
├────────┼──────┼────────┼──────────────────────────────┤
│ 练气   │ 120年│ 1~9层  │ 灵力积累满即可              │
│ 筑基   │ 200年│ 初/中/后│ 需筑基丹 + 灵力 + 悟性考验  │
│ 金丹   │ 500年│ 1~9转  │ 需凝丹 + 心境 + 特定材料    │
│ 元婴   │ 1000 │ 初~后  │ 心魔劫 + 元婴凝聚           │
│ 化神   │ 2000 │ 初~后  │ 天劫(雷劫) + 领悟一道       │
│ 炼虚   │ 5000 │ 初~后  │ 天劫 + 空间法则             │
│ 合体   │ 1万年│ 初~后  │ 重劫 + 法则融合             │
│ 大乘   │ 5万年│ 初~后  │ 仙劫(极难)                  │
│ 渡劫   │ 飞升 │ ─     │ 飞升劫(终极挑战)            │
└────────┴──────┴────────┴──────────────────────────────┘
```

### 6.2 功法分类

```
功法类型：
┌────────────────────────────────────────────┐
│ 📖 修炼功法  │ 影响修炼速度和基础属性       │
│              │ 每次只能主修1部              │
│              │ 例：《太上感应篇》《血煞经》 │
├──────────────┼─────────────────────────────┤
│ ⚔️ 战斗功法  │ 战斗中使用的技能             │
│              │ 装备在主动/被动槽            │
│              │ 例：烈火掌、御剑术           │
├──────────────┼─────────────────────────────┤
│ 🏃 身法/遁术 │ 移动和逃跑                   │
│              │ 影响实时模式移速和逃跑概率   │
├──────────────┼─────────────────────────────┤
│ 🛡️ 护体功法  │ 被动防御                     │
│              │ 减伤、抗性、反伤             │
├──────────────┼─────────────────────────────┤
│ 🔮 辅助功法  │ 炼丹加成、悟道加成等         │
└──────────────┴─────────────────────────────┘

功法品质：凡品 < 黄品 < 玄品 < 地品 < 天品 < 仙品
```

### 6.3 功法学习机制

```
获取途径：
  · 宗门传授（加入宗门后可学）
  · 秘境获取（探索遗迹）
  · NPC传授（好感度足够）
  · 自创（高悟性 + 悟道）
  · 拍卖会/交易

熟练度系统：
  初学(1级) → 熟练(5级) → 精通(10级) → 大成(15级) → 圆满(20级)
  
  · 熟练度越高 → 威力越大、消耗越低
  · 通过使用(战斗中)和参悟(闭关)提升
```

### 6.4 自创功法系统（高阶玩法）

```
条件：化神境以上 + 对应大道悟透

流程：
  1. 选择基础模板（攻/防/辅/遁）
  2. 选择核心属性（五行/异变）
  3. 注入已悟的"道韵"
  4. 调整参数（威力/消耗/特效）
  5. 命名 → 完成

→ 自创功法可以传授给弟子/道侣
→ 可以作为宗门传承
```

### 7.2 悟道方式

```
悟道途径（多管齐下）：
┌──────────────────────────────────────────┐
```

## 11. 渡劫与寿命系统

### 11.1 寿命机制（核心压力源）

```
寿命消耗：
  · 日常活动：每年自然消耗1年(基础)
  · 闭关修炼：加速消耗（1次=数年）
  · 重伤恢复：消耗额外寿命
  · 某些功法：以寿命换实力(魔道)

寿命延长：
  · 突破境界 → 寿命上限大幅提升
  · 服用延寿丹 → +固定年限（有极限）
  · 特殊功法 → 减缓衰老
  · 悟时间大道 → 寿命消耗减缓

寿命归零：
  · 角色自然死亡
  · 可选择"转世重修"（保留部分悟道进度）
```

### 11.2 渡劫系统

```
化神境及以上，每次突破需渡劫：

劫的类型：
┌──────────┬───────────────────────────────────┐
│ 雷劫     │ 最常见，连续雷击，需硬抗          │
│ 心魔劫   │ 心境考验，战胜内心幻象            │
│ 业火劫   │ 焚烧业力，善恶值影响难度          │
│ 天罡劫   │ 物理打击，防御考验                │
│ 仙劫     │ 大乘→渡劫，极难，需大量准备       │
└──────────┴───────────────────────────────────┘

渡劫准备：
  · 准备渡劫丹（回复/护体）
  · 布置阵法（减伤）
  · 选择地点（灵气浓郁处）
  · 邀请护法（道侣/弟子助阵）

渡劫结果：
  · 成功 → 境界提升，寿命暴涨
  · 失败 → 
    - 轻则重伤(损失大量寿命)
    - 重则身死道消(可转世重修)
```

### 11.3 转世重修系统（死亡不是终点）

```
触发条件：寿命归零 / 渡劫失败 / 战死

转世保留：
  · 悟道进度（部分，看境界）
  · 自创功法（化为传承玉简）
  · 部分气运
  · 前世记忆（影响事件选择）

转世损失：
  · 全部境界
  · 全部属性
  · 大部分物品
  · 人际关系（除非有特殊道具）

→ 鼓励玩家"多周目"体验不同路线
→ 每次"死"都有价值，不是纯惩罚
```

```gdscript
var participants: Array[Character] = []
var turn_queue: Array = []          # 行动顺序队列

    participants = player_team + enemy_team
    _calculate_turn_order()
    _enter_battle_scene()

func _calculate_turn_order():
    # 按速度排序，ATB时间轴
    turn_queue = participants.duplicate()
    turn_queue.sort_custom(_compare_speed)

func _compare_speed(a, b):
    return a.attributes.speed > b.attributes.speed

func execute_action(character: Character, action: BattleAction):
    match action.type:
        "attack":  _execute_attack(character, action)
        "skill":   _execute_skill(character, action)
        "item":    _use_item(character, action)
        "defend":  _defend(character)
        "flee":    _attempt_flee(character)
    
    _check_dao_resonance(action)
    _check_battle_end()
```

## 📌 总结

本设计文档以**《觅长生》的回合策略深度** + **《鬼谷八荒》的开放世界自由度**为核心，通过以下设计实现融合：

| 设计点 | 融合方式 |
|--------|---------|
| **混合制战斗** | 实时探索（鬼谷）+ 回合策略（觅长生），兼顾操作感和策略深度 |
| **道韵共鸣** | 将觅长生的功法连招扩展为元素反应系统，增加战斗变化 |
| **寿命驱动** | 鬼谷的寿命压力 + 觅长生的多系统选择，营造"时间稀缺"体验 |
| **多途径悟道** | 不仅靠闭关，战斗/探索/社交都能悟道，鼓励多元玩法 |
| **副职业闭环** | 五大副职业互相供给，形成完整生产链 |
| **NPC生态** | 鬼谷的活NPC + 觅长深的深度关系（师徒/道侣/传承） |
| **转世重修** | 降低死亡挫败感，鼓励多周目探索不同路线 |

> **核心理念：让每一局修仙之旅都是独一无二的。**
> 时间有限，选择无限，道无穷尽。
