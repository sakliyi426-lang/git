## 1. 寿命系统调整

### 1.1 调整原则

适度淡化寿命压力，从"倒计时压力"变为"长期规划参考"，保留存在感但增加容错空间。

### 1.2 具体调整

| 调整项 | 原设计 | 新设计 |
|--------|--------|--------|
| 各境界基础寿命 | 练气120年,筑基200年... | **整体上浮约60%**（练气200年,筑基320年...） |
| 闭关时间消耗 | 1次=1~10年 | **1次=数月至3年** |
| 探索时间消耗 | 1区域=数月~1年 | **1区域=数周~数月** |
| 延寿途径 | 延寿丹+特殊功法 | +傀儡延寿术(傀儡替身承受衰老)、+灵植延寿丹原料更容易获取 |
| 时间大道效果 | 寿命消耗减缓 | **寿命消耗减缓效果翻倍** |

### 1.3 轮回传承系统

寿尽后角色死亡，触发轮回：

**继承项**（保留）：
- 前世记忆（解锁特殊对话/事件选项）
- 全部功法熟练度
- 全部悟道进度

**封印遗产**：
- 寿尽前消耗**阵道能力**+特殊材料，将物资封印在指定地点
- 阵法等级越高，封印越稳定，越不易被NPC发现
- 新角色达到一定境界/到达特定位置后，可感知前世封印方位
- 可前往发掘，继承前世遗产

**损失项**（重置）：
- 境界清零
- 全部属性重置
- 物品丢失（除非封印）
- 人际关系清零

**"宿命感应"机制**：
- 高悟性角色能隐约感知前世封印的大致区域
- 随着境界提升，感应越来越清晰

## 2. 傀儡炼制副职业（第6副职业）

### 2.1 职业定位

独立副职业，但依赖**炼器**产出部件材料，形成"部件→组装"协同关系。

### 2.2 炼制流程

```
获取傀儡图纸 → 准备部件（炼器产出） → 炼制核心（决定品质） → 组装成型 → 激活启灵（炼丹产出启灵丹）
```

### 2.3 材料体系

| 材料类型 | 来源 | 用途 |
|---------|------|------|
| 主体框架 | 炼器（矿石/灵木） | 傀儡基础机体 |
| 核心阵盘 | 阵法（炼制） | 决定品质上限和基础属性 |
| 灵络 | 灵植（灵丝/魂丝） | 决定灵性（AI智能程度） |
| 启灵丹 | 炼丹 | 激活新傀儡 |
| 修复膏 | 炼丹 | 恢复耐久 |
| 精炼液 | 炼丹 | 提升品质 |

### 2.4 品质体系

```
凡品 < 黄品 < 玄品 < 地品 < 天品 < 仙品
```

品质决定：
- 基础属性范围
- 可搭载模块数量
- 灵性上限（AI智能化程度）
- 耐久上限

### 2.5 熟练度

1~10级，每级解锁更高级品质 + 更复杂图纸。

## 3. 傀儡属性与模块

### 3.1 基础属性

- **气血**：生命值
- **灵力**：技能能量
- **攻击**：物理伤害
- **防御**：减伤
- **速度**：ATB行动顺序

### 3.2 专属属性

| 属性 | 说明 |
|------|------|
| **承载** | 决定可搭载模块数量（品质+境界影响） |
| **灵性** | AI智能化程度，影响自动战斗/派遣效率 |
| **硬度** | 耐久上限，影响维修频率和成本 |

### 3.3 模块槽

| 槽位 | 数量 | 来源 |
|------|------|------|
| 武器模块 | ×1 | 炼器产出 |
| 防具模块 | ×1 | 炼器产出 |
| 功能模块 | ×2（随品质解锁更多） | 炼器/符箓产出 |

### 3.4 模块类型

```
武器模块：机关飞刃、火灵炮、冰霜喷射器...
防具模块：铁壁屏障、灵力护盾、反伤甲...
功能模块：撒网束缚、治疗灵光、自爆装置、探测雷达...
符箓模块：一次性自动触发BUFF（符箓副职业产出）
```

## 4. 傀儡战斗系统

### 4.1 战斗规则

- 傀儡作为**独立单位**参与ATB回合制战斗
- 最多**同时上场2个**傀儡
- 傀儡行动不消耗灵力，消耗**耐久度**
- 耐久归零 → 傀儡报废（可拆解回收部分材料）

### 4.2 默认挂机AI

玩家预设行为优先级（可随时切换）：

```
1. 均衡型：攻守平衡，自动选择最优行动
2. 攻击型：优先输出，集火低血量目标
3. 防御型：优先保护主人，承担伤害
4. 辅助型：使用BUFF/治疗模块
```

### 4.3 远程操控模式

- 战斗中可**切换操控到傀儡**，精细化操作（指定目标、使用特定模块技能）
- 操控模式下傀儡不消耗额外耐久，但消耗玩家**神识**
- 玩家操控傀儡期间，角色本体**自动防御**

### 4.4 耐久与修复

- 战斗消耗耐久，归零则报废
- 修复需消耗灵石 + 对应部件材料
- 在洞府傀儡工坊或城镇傀儡铺修复

## 5. 傀儡探索派遣系统

### 5.1 派遣操作

- **随时随地**通过界面下达派遣指令
- 派遣不消耗玩家时间，傀儡独立行动
- 选择条件：目标区域必须是已探索过的区域

### 5.2 派遣参数

| 参数 | 选项 |
|------|------|
| 目标区域 | 世界地图大区 / 秘境（已探索） |
| 时长 | 短期（数天）/ 中期（数月）/ 长期（数年） |
| 目标类型 | 资源采集 / 情报侦察 / 秘境探路 |

### 5.3 派遣结果

| 结果 | 说明 |
|------|------|
| 成功带回 | 灵草、矿石、妖兽材料、随机物品 |
| 遭遇战斗 | 消耗耐久，可能受损，严重则报废逃回 |
| 发现特殊 | 低概率发现隐藏秘境入口/遗迹坐标 |
| 失联 | 高风险区域有概率失联（可派遣更强傀儡寻找） |

### 5.4 远程操控探索

- 可随时**切换视角到傀儡**，手动操控其在派遣区域探索
- 操控期间角色本尊时间照常流逝（不受影响）
- 消耗神识，不足则自动切回挂机

### 5.5 信息共享

- 傀儡探索过的区域对角色永久可见（解锁迷雾）
- 遭遇过的敌人记入图鉴
- 高级傀儡绘制简易地图，本尊后续探索效率提升

## 6. 傀儡自动化生产

### 6.1 挂机炼制

- 傀儡可配置炼丹炉/炼器台，自动炼制**低等级**丹药和武器
- 低等级定义：当前玩家境界以下1~2个等级的配方
- 需要玩家提供原料+丹方/器谱
- 产出品质受傀儡品质+灵性+对应副职业熟练度影响
- 可设置生产队列（如"持续炼制回血丹直到原料耗尽"）

### 6.2 辅助灵植

- 傀儡自动照料灵田：浇水、除虫、施肥、收获
- 多傀儡同时管理多块灵田
- 高级傀儡感知灵植状态，自动处理病害/成熟收割
- 收获灵草可供挂机炼丹使用

### 6.3 自动化循环示例

```
傀儡A：探索派遣 → 带回矿石/灵草
傀儡B：洞府挂机炼器 → 消耗矿石产出部件
傀儡C：洞府挂机炼丹 → 产出丹药供战斗
傀儡D：照料灵田 → 收获灵草供傀儡C炼丹
玩家只需：提供图纸配方 + 定期补充原料 + 收取成品
```

## 8. 与现有五大副职业协同

```
炼器 ──产出部件──▶ 傀儡 ──辅助战斗──▶ 探索/秘境
  ▲                    │                     │
  │                    │ 派遣探索带回原料      │
  │                    ▼                     ▼
  │                 灵植 ◀──── 炼丹 ────▶ 战斗丹药
  │                                   
  └── 阵盘/防护 ◀── 阵法
```

| 副职业 | 协同内容 |
|--------|---------|
| **炼器** | 主体框架、武器/防具模块、精密机关构件 |
| **阵法** | 核心阵盘（品质上限）、傀儡传送阵（远程操控信号强化） |
| **灵植** | 灵木/灵丝（机体材料）、灵髓果（能源） |
| **炼丹** | 启灵丹（激活）、修复膏（耐久）、精炼液（品质提升） |
| **符箓** | 符箓模块（可嵌入功能槽，一次性自动触发） |
